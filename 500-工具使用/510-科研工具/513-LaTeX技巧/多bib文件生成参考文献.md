# 多bib文件生成参考文献

> tags: #LaTeX; #BIBTeX;

## TL; DR

```tex
\bibliography{reference,references/file1,references/file2}
```

## 详细流程

将对应的 `bib` 文件名用逗号间隔添加至 `\bibliography{}` 当中，注意逗号之间不必加空格间隔。
